# Replication Code for: "A Joint Model for Multivariate Longitudinal Biomarkers and Survival Outcomes with a Fast Kalman Filter"

**Authors:** Mingzhao Hu, Ya Luo, Yuedong Wang

---

## Overview

This repository contains R code for the real-data analysis and simulation studies in the manuscript.

The repository currently has two roles:

1. **Real-data analysis code archive.**  
   The scripts for fitting the joint model and generating the real-data figures/tables are included. The restricted dialysis data objects are not included in this public repository.

2. **Simulation replication.**  
   The public files are sufficient to reproduce the supplemental simulation summaries based on uploaded simulation parameter-estimate objects.

The real-data figures and tables require restricted data files and precomputed fitted/bootstrap objects. The simulation tables can be reproduced from the uploaded simulation objects after placing the files in the paths expected by the scripts.
